## 1. 소상공인시장진흥공단 (semas.or.kr)

### A. 메뉴 구조

**GNB (6개 대분류)**
| 1Depth | 2Depth (주요) |
|--------|---------------|
| 알림마당 | 공지사항, 입찰정보, 양도담보물공매, 사업공고, 유관공고, 채용정보, 보도/설명, 안전게시판 |
| 지원사업 | 4개 카테고리 분류: 성장형 소상공인 육성(4), 상권·시장 조성(5), 자금부담 완화(3), 정책서비스 강화(2) |
| 정보공개 | 정보공개 안내, 사전정보공표, 공공데이터 등 |
| 경영공시 | 경영실적, 재무정보, 사업실적 등 |
| 국민소통 | 고객소통, 민원, FAQ 등 |
| 공단소개 | 기관개요, 조직, 오시는길 등 |

**퀵메뉴/바로가기**

- 상단 빠른 접근 카드 4개: 소상공인 정책자금 / 지식배움터 / 소상공인24 / 소상공인365
- 전국 소상공인지원센터 (새출발지원센터) 12개 지역별 바로가기
- 확인서/신고 바로가기 단축링크
- 외부 연계: 중소벤처기업부, Alio, 온누리상품권 등

### B. 메인 페이지 정보 알림 구조

**공지/뉴스 배치: 탭형 (4개 탭)**

- 사업공고 / 공지사항 / 유관공고 / 직원채용
- 각 탭은 슬라이드 형식으로 최대 4개 항목 노출
- 반응형: 1100px 이하에서 2개 항목으로 축소

**배너/팝업**

- 헤더 팝업 (topPop)
- 메인 팝업 2종 (pop_101, pop_102) - "오늘하루 보지 않기" 옵션
- 슬라이드 배너: dots 네비게이션, 6초 간격 자동재생

**서비스 카드**

- 소식알림 슬라이드: 화살표 버튼 + 페이지 정보
- 기관우수사례: 4개 아이템 (모바일 1~2개)
- 지원사업 알아보기: 2개 아이템 (모바일 1개)
- SNS 채널 통합 (유튜브, 페이스북, 인스타, 블로그) 피드 섹션

### C. 사용자 동선

**정책자금 정보 접근 경로**

1. 메인 > 상단 퀵카드 "소상공인 정책자금" 클릭 (1클릭)
2. 또는 GNB > 지원사업 > 소상공인 자금부담 완화 (2클릭)

**상담 신청까지 클릭 수: 3~4클릭**

- 메인 > 정책자금 > 신청 페이지 > 회원 로그인/신청서 작성

**자격 확인 → 신청 프로세스**

- 온라인 신청 사이트(ols.semas.or.kr)에서 진행
- 직접대출: 온라인 접수 → 현장실사(사업성/시장성/경영능력/신용도) → 대출 실행
- 대리대출: 확인서 신청 → 보증기관 접수 → 보증서 발급 → 금융기관 대출

### D. 특징적 UI 패턴

- **탭 기반 공지 영역**: 4개 탭으로 콘텐츠 분류, 슬라이드 전환
- **호버 효과**: 이사장앨범 mouseover 시 콘텐츠 박스 표시
- **반응형 슬라이드**: 해상도별 아이템 수 자동 조절
- **채팅 버튼**: 모바일/데스크톱 구분 반응형 팝업
- **세션 연장 알림**: 60초 전 로그아웃 예고 팝업

## 2. 서울신용보증재단 (seoulshinbo.co.kr)

### A. 메뉴 구조

**GNB (추정 5~6개 대분류)**
| 1Depth | 주요 내용 |
|--------|-----------|
| 보증안내 | 신용보증 상품, 보증절차, 보증료 안내 |
| 경영지원 | 세무·노무·마케팅·디지털전환 컨설팅 |
| 창업지원 | 창업보증, 창업컨설팅, 창업교육 |
| 재단소개 | 기관소개, 지점안내(17개 지점), 경영공시 |
| 알림마당 | 공지사항, 사업공고, 보도자료 |
| 고객서비스 | 상담예약, 이용내역 조회, 채무잔액확인서 |

**퀵메뉴/바로가기**

- 디지털 종합지원센터 (상담예약, 이용내역 조회, 채무잔액확인서 발급)
- 온라인 신청 시스템
- 소상공인 마이데이터 (mydata.seoulshinbo.co.kr)

### B. 메인 페이지 정보 알림 구조

- 공지사항/뉴스 배치 (리스트형 추정)
- 주요 서비스 바로가기 카드: 신용보증, 경영지원, 창업지원 등
- 교육 프로그램 안내 배너
- 재단 정책/성과 정보 섹션

### C. 사용자 동선

**보증 신청 프로세스 (4단계)**

1. 상담 및 보증접수 (온라인/방문/전화)
2. 신용조사
3. 보증심사
4. 보증서 발급

**상담 신청까지 클릭 수: 2~3클릭**

- 메인 > 디지털종합지원센터 > 상담예약
- 또는 메인 > 보증안내 > 온라인 신청

**종합상담창구 특징**

- 1회 방문으로 보증 + 창업·재창업 + 세무·노무 + 판로개척 + 디지털전환 원스톱 지원
- 전 지점을 '종합지원센터'로 전환

### D. 특징적 UI 패턴

- **종합지원센터 모델**: 금융지원 중심에서 '소상공인 종합병원' 모델로 전환
- **모바일 앱 연동**: 상담 예약, 보증신청, 서류 제출 모바일 지원
- **마이데이터 서비스**: 별도 서브도메인으로 마이데이터 플랫폼 운영
- **지점 찾기**: 서울시 17개 지점 위치 안내

4. 융자 신청 (융자신청서 제출)
5. 기업평가 (현장방문 또는 비대면)

## KPEC 사이트 도입 추천 패턴

### 1. 메뉴 구조 (GNB)

**추천: 5개 대분류 + 퀵메뉴 카드형**

```
정책자금안내 | 상담신청 | 성공사례 | 자료실 | 센터소개
```

- 소진공의 4개 카테고리 분류 방식 차용 (자금 유형별 분류)
- 기보의 이원화 구조 참고: 정보 제공 영역과 신청/업무 영역 분리

### 2. 메인 페이지 정보 알림

**추천: 소진공식 탭형 공지 + 신보식 서비스 카드**

- **상단 히어로**: 핵심 메시지 + CTA 버튼 (정책자금 상담 바로가기)
- **퀵 서비스 카드 (4개)**: 정책자금 조회 / 자격 확인 / 상담 신청 / 성공사례
- **공지 탭 영역 (3탭)**: 공지사항 / 사업공고 / 뉴스
- **배너존**: 주요 사업 안내, 신규 정책자금 알림

### 3. 사용자 동선

**추천: 서울신보식 원스톱 + 기보식 추천 시스템**

- **자격 확인 → 상담 신청 2클릭 이내**로 설계
- 메인 > "정책자금 자격확인" 버튼 > 간단 설문 > 맞춤 상품 추천 + 상담 신청
- 기보의 "추천서비스" 패턴 도입: 업종/규모/자금용도 입력 → 자동 매칭

### 4. 특징적 UI 패턴

**도입 우선순위**

| 순위 | 패턴                 | 참고 사이트     | 적용 위치          |
| ---- | -------------------- | --------------- | ------------------ |
| 1    | 정책자금 맞춤 추천   | 기보 추천서비스 | 메인 CTA           |
| 2    | 탭형 공지 게시판     | 소진공 4탭      | 메인 하단          |
| 3    | 신청 프로세스 시각화 | 신보 4단계      | 신청 안내 페이지   |
| 4    | 원클릭 상담 신청     | 기보 원클릭     | 모든 페이지 플로팅 |
| 5    | 자격 조건 표/카드    | 소진공 카테고리 | 정책자금 상세      |
| 6    | FAQ 아코디언         | 신보 소통참여   | 자료실             |
| 7    | 비대면 플랫폼 연계   | 신보 ON-Biz     | 상담/신청          |
| 8    | 성공사례 카드 갤러리 | 소진공 우수사례 | 메인 + 별도 페이지 |

### 5. KPEC 차별화 포인트

1. **정책자금 매칭 엔진**: 기보의 추천서비스를 더 간소화하여 3가지 질문(업종/규모/자금용도)으로 맞춤 정책자금 추천
2. **상담 예약 원스톱**: 서울신보의 종합상담창구 모델 + 온라인 예약 시스템
3. **신청 프로세스 시각화**: 신보의 4단계 프로세스를 스텝 인디케이터로 시각화
4. **모바일 최적화**: 모든 벤치마크 사이트가 데스크톱 중심이므로, 모바일 퍼스트 접근으로 차별화
5. **실시간 상담**: 소진공의 채팅 버튼을 발전시킨 실시간 상담 채널 (카카오톡 연동 등)
